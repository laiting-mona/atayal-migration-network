

**裂解與重組：以社會網絡分析探索泰雅族歷史遷徙網絡的結構變遷**

*Fragmentation and Reconfiguration: Exploring the Structural Transformationof Atayal Historical Migration Networks through Social Network Analysis*

組員：賴亭穎、黃凡嘉

# **1\. 研究背景**

泰雅族是臺灣原住民族中分布領域最廣的民族，橫跨新北、桃園、新竹、苗栗、臺中、南投、宜蘭等山區。族群內部分為賽考列克與澤敖列兩大亞群，分別認定南投仁愛鄉瑞岩與新竹五峰大壩尖山為發源地。這種「共同祖源、多路線擴散」的遷徙敘事，具有鮮明的網絡拓撲意涵：從少數起源節點向外輻射，沿河谷、山稜形成分支路徑。

泰雅族的遷徙史可粗分為三個結構性斷裂的時期。第一個時期是十七世紀末至十九世紀末的自發擴散期。十八世紀漢族由西部平原向淺山拓墾，加上平埔族移住埔里，泰雅族人為尋找狩獵與耕地，從南投北港溪上游向北、向東擴散，陸續抵達苗栗、新竹、桃園、宜蘭與新北山區。此階段以血緣親族為單位，沿溪流中上游建立散村式部落，部落間透過聯盟 mulaxen galang 維繫防禦性的區域網絡。

第二個時期是 1895 至 1945 年的殖民強制移住期。日治時期的理番政策對泰雅族的空間分布造成劇烈重組。集團移住由 1919 年起試辦，1930 年霧社事件後正式大規模推動，目的在於開發山地資源並方便管理。布農族與泛泰雅族群被視為最難管理的對象，首當其衝成為移住重點。這一階段的遷徙不再循自然地理邏輯，而是由殖民行政力量重新配置，打破原有地緣互動網絡，甚至將文化上不相關的部落強制合併至同一聚居地。歷史文獻明確記載，集團移住之安排「不符原有地緣互動性」。戰後國民政府延續此政策，1950 至 1971 年間計有 46 個村落、近 15,978 人遷至新聚落。

第三個時期是 1945 年至今的戰後延續與都市化遷移期。隨著經濟結構轉型，大量原住民青壯年人口流入都會區，部落間的連結方式從地理鄰近轉向跨區域的親族與同鄉網絡。

在社會組織層面，泰雅族傳統部落以血緣親族為基礎，配合祭祀、狩獵、負罪牲團、勞動等功能性組織。部落對外透過聯盟 mulaxen galang 保持聯繫、結盟防守。從家戶到 gaga、到部落、到部落聯盟的多層結構，本身即是嵌套式的社會網絡系統，適合以 SNA 多層次架構刻畫。然而泰雅族部落間的互動並非僅有合作與聯盟。日本學者岡松參太郎曾指出，北部原住民族以泰雅族為主，頭目實力較弱，社會組織較為平等，部落間爭執最常以出草 mgaga 解決，比南部階層性強的族群更容易產生武力衝突。日治時期殖民者更利用此一特性，以「味方番」策略分化部落、使其自相殘殺。當代田野調查中亦有泰雅獵人表示，泰雅族是現代社會中各原住民族裡部落間互動最少的族群。這些觀察是否反映在當代遷徙網絡的結構特徵中，是本研究希望驗證的重要問題。

在學術脈絡上，廓守臣《泰雅族的文化：部落遷徙與拓展》是最早系統重建泰雅遷徙史的民族誌。近年 TICD 與 TIPsMigDynamics 等數位資料庫提供了結構化的部落位置與遷徙動態記錄，使量化網絡分析成為可能。國際上，Mills et al. 以 dynamic SNA 分析北美原住民古代遷徙，證明網絡連結性既先於遷徙發生，也是遷徙的結果。然而臺灣原住民族的遷徙至今尚無 SNA 系統性分析，本研究意在填補此空白。

# **2\. 研究動機**

本研究的動機來自多個層面。首先，組員曾於宜蘭南澳參加地方創生實習，隨泰雅獵人上山學習陷阱製作、獵區分野、族語與傳說，並閱讀《南澳泰雅人》田野紀錄，塑成對泰雅部落變遷的興趣。

其次，集團移住本質上是一次由上而下的網絡重組，強制切斷既有地緣連結、將散居部落合併移至指定地點。然而這種結構性斷裂對網絡拓撲的具體影響，至今缺乏量化描述。本研究將比較不同時期的網絡指標，具體呈現殖民政策的結構性影響。

第三，近年 TICD、TIPsMigDynamics 等資料庫的建置，加上中研院「原住民部落遷徙與分布」GIS 資料集的公開，使過去散落在日治檔案、口述傳統與民族誌中的遷徙紀錄得以結構化為可計算的邊列表。本研究欲善用這些資源，示範歷史民族誌與網絡科學的跨域方法論。

第四，近年 Science 等期刊刊出北美原住民強制遷徙的量化研究，但東亞南島語族的遷徙網絡在國際文獻中仍嚴重缺席。臺灣作為南島語族可能原鄉，本研究可與國際學術社群形成方法論對話。

Granovetter 指出弱連結在跨群體資訊傳遞中扮演關鍵角色。泰雅族遷徙中，聯盟內部為強連結，跨流域遷徙則可能仗賴弱連結。透過分析 bridge edges 與 structural holes 的分布，可檢驗此一理論在歷史遷徙中的適用性。

最後，田野中泰雅獵人表示泰雅族是各原住民族中部落間互動最少的族群。本研究將此轉化為可檢驗的網絡假說：若確實如此，遷徙網絡應呈現較低的 clustering coefficient、較高的 modularity、較多的 structural holes；若擴展至跨族群比較，泰雅族與其他族群的指標應有顮著差異。

# **3\. 研究目的**

本研究旨在結合台灣原住民族部落開放資料庫（TICD）與台灣原住民各族群遷徙動態（TIMD / TIPsMigDynamics），以社會網絡分析（Social Network Analysis, SNA）與圖論方法建構泰雅族當代遷徙網絡，探討泰雅族在當代人口流動中所呈現的空間連結、社群結構與關鍵節點角色。由於 TIMD 的資料是依人口動態資料建構的點對點遷徙動態，並且可區分遷出地與遷入地，因此本研究希望透過網絡方法，將原本分散於人口動態與部落資料中的遷徙關係，轉換成可分析的有向連結結構，以理解泰雅族遷徙網絡的整體型態。

具體而言，本研究有三個核心目的。第一，辨識泰雅族遷徙網絡中的核心節點與橋接節點，也就是哪些部落或地點在遷徙流中具有較高的遷入吸引力、較強的遷出擴散性，或位於不同區域之間的重要中介位置。第二，分析泰雅族遷徙網絡是否形成若干穩定的社群結構，例如北部山區、淺山地帶、都會周邊或跨縣市生活圈之間的聚集現象。第三，從整體網絡層次檢視泰雅族遷徙是否呈現明顮的中心化、群聚化或核心邊陳特徵，藉此理解當代人口流動與部落空間分布之間的關係。

本研究選擇聚焦泰雅族，除了因為泰雅族是台灣原住民族中分布範圍最廣的民族之一，橫跨新北、桃園、新竹、苗栗、台中、南投與宜蘭等地，也因其分布跨越多個山地與流域空間，特別適合觀察遷徙網絡中不同區域之間的連結強度與橋接關係。透過這樣的研究設計，本研究希望從「部落—地點—遷徙流」的網絡角度，補充過去較偏重歷史敘事或區域描述的研究方式，提供一個更量化且可視化的原住民族遷徙分析架構。

# **5\. 研究方法**

本研究將以 TIMD / TIPsMigDynamics 作為遷徙網絡的主要資料來源，並以 TICD 作為部落節點與背景資訊的輔助資料。TIMD 資料是根據人口動態資料所建立的點對點遷徙動態，涵蓋初級、回流與連續遷徙，並以個人遷徙流為基礎。TICD 提供眾多部落的量化資料，包括總人口數、性比率、教育程度、年齡結構、婚姻狀態、各族群人口比例，以及部落人口重心座標，可作為節點定義與部落背景分析的重要基礎。

在網絡建構上，本研究將採用有向加權圖作為主要模型。原因是遷徙資料本身具有明確的起點與終點，因此 A 地遷往 B 地，與 B 地遷往 A 地，在社會意義與人口流向上並不相同。節點可先設定為人口動態資料中的地點單位；若資料整合可行，則再透過 TICD 的部落位置與人口重心資訊，將節點進一步對應到泰雅族部落。邊則定義為從起點到終點的遷徙連結；邊的權重則定義為在特定期間內，從 i 地遷往 j 地的人數或次數。由於原始資料說明點位是村里人口重心，因此研究實作上將以地點層級建立網絡，再視需要映射回部落層級。

在時間處理上，本研究將採用時間切片方式，按年或按前後期切分，建立多個時段的遷徙網絡，以比較不同時段之間的結構變化。

在分析指標上，本研究將藉由 in-degree 與 out-degree，以分別辨識主要遷入中心與主要遷出來源；計算 betweenness centrality 以找出位於不同區域或社群之間、扮演橋接角色的節點；並計算 clustering coefficient 與進行 community detection，觀察泰雅族遷徙網絡是否形成高密度的局部群聚與穩定社群。若分析結果顮示某些節點同時具有高遷入量與高橋接性，則這些節點可被視為泰雅族當代遷徙網絡中的 hub 或 bridge。

最後，在研究對象篩選上，本研究將聚焦於泰雅族分布較集中的區域，包括新北、桃園、新竹、苗栗、台中、南投與宜蘭等中北部山區與相關生活圈。研究流程上，將先從遷徙動態資料中篩出泰雅族相關遷徙流，再結合 TICD 的部落資料建立節點資訊，最後透過網絡分析與視覺化呈現，解釋泰雅族遷徙網絡中的核心結構與區域連結。

# **6\. 預期成果**

本研究預期建立一套以實際公開資料為基礎的泰雅族當代遷徙網絡模型。這個模型將原先對於遷徙現象的抽象描述，以「從哪裡移出、往哪裡移入、哪些地點互動最頻繁」的量化方式建構方向性與強度，轉換為可視覺化與可衡量的社會網絡結構。

其次，本研究預期能辨識出泰雅族遷徙網絡中的幾類重要節點。第一類是主要遷入中心，也就是吸納較多人口流入的節點；第二類是主要遷出來源，代表向外移動較頻繁的地點；第三類是橋接節點，也就是連結不同區域、不同社群或不同生活圈的重要中介地。這些結果可以幫助我們更具體地理解，泰雅族的當代人口流動並非隨機散布，而可能呈現明顮的中心、走廊與連結帶。

最後，本研究預期能找出泰雅族遷徙網絡中的社群結構與區域群聚現象。若 community detection 顮示網絡中存在若干穩定群集，則可進一步討論這些群集是否對應到地理區位、山區與淺山的互動帶、跨縣市生活圈，或部落分布與現代居住空間之間的延續關係。若網絡呈現高群聚或核心邊陳結構，也能進一步說明泰雅族在當代遷徙上是否具有明顮的中心化特徵。
